Mixar is a mobile application where the user can go between musicians in an ensemble and listen to the specific instrument played. It was developed in the course Multimodal Interactions and Interfaces at KTH and was done in Unity using Augmented Reality (AR). The instruments, or musician, can be placed differently using cards. This can make the user listen to an ensemble in a bigger or smaller room and listen to how the spatial sound of the ensemble will be like.
